# PyPair

PyPair is a statistical library to compute pairwise association between any two variables. A [reasonable taxonomy](https://cyfar.org/types-variables) of variable types in statistics is as follows.

* categorical
  * binary
  * nominal
  * ordinal
* continuous
  * interval
  * ratio
